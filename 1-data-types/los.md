Learning Objectives

* Explain what a sum type is

* Explain what a product type is

* Explain what a functor is

* Explain how to encode data types using functors, sums, and products

* Encode simple data types as functors and give their recursion schemes in Haskell

* Apply folds to implement basic recursive functions

* Apply unfolds to implement basic corecursive functions
